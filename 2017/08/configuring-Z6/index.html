上船溺水指南 - 给船安装沉没系统

2017-08-17

整个过程真是一波三折啊……带上呼吸器继续潜水【滑稽】

首先解决 windows 。

windows part

  1. 安装时用 diskpart 分区， 4k 对齐（然鹅 windows 还是会自动生成恢复分区）
  2. 用傲梅助手分区，砍掉recovery分区——这样做之后碰巧一段时间开机总是要扫描某个磁盘，一开始以为是系统分区，后来才发现是 700G+ 的数据分区……好吧——稍微给 SSD 紧张的占用喘口气。
  3. 升级红石更新，磁盘清理删除 windows.old
  4. 用 dism 添加组策略工具
  5. 设置 uac
  6. 各种驱动，常用工具……

arch part
第一天
一开始觉得轻车熟路，结果第一遍忘了装 bootloader ……然后依次忘了 dialog，wpa_supplicant……好几次连不上网络。

主要参考 这篇 。

测速随便挑了一个 fetchmirrors ，生成的 mirrorlist 还算可以。

lightdm 启动不了非常绝望，发现没装 greeter（并不是）

第二天
xf86-video-intel

systemctl enable/start acpid

结果还是不行。

后来发现 startx 不了是因为没装 twm ，装好重启之后 surprised really ，跳出来 xterm 啥的 一股 windows 3.1 的气息 ，然后开 lightdm 和 i3 就好了。

第四天
蓝牙自启动，见 Bluetooth ；另外设置信任设备。

设置 i3-bar 的时候发现亮度不工作……搞了半天是没装 bc 。

rsync 转移备份数据。

安上 fcitx-gtk2 输入法终于好了！

U 盘只读是因为没用上 ntfs-3g 。

调度器设为 noop 。

爱惜 SSD 从我做起。人在做天在看，用 profile-sync-daemon 保平安。
