上船溺水指南 - 神船 Z6-KP5D1 开箱

2017-08-13
laptop

这两天船不是搁浅了就是烧焦了——真是令人窒息的操作。

好吧，东西是大约一星期前收到的，玩了几天看门狗终于定下心思开始写开箱 post 了；说起来，神船不仅质量让人担忧，在软件配置上也频频让我捏一把汗——当然那是后话。

关于配置的问题，之前考虑要不要买 i7-6700HQ 的机子，后来考虑到超线程带给的提升并不是很多，而且散热未必受得住，加上内存规格等问题，就没有上 i7 。所以先在这艘船是长这样的：

CPU : i5-7300HQ
GPU : Nvidia GTX 1050
MEM : 8 GB DDR4 2133MHz * 1
HDD : 1TB 5400RPM （貌似是日立？）
SSD : 128GB TLC (Lenovo)
Screen : 1080p TN
Battery : ~45Wh

但是 Z6 的模具散热似乎确实不好，现在就很烫手 皱眉 。

首先，拆开来东西是这样的：

我在下单的时候跟卖家说要 A 面没有图案的……但是他们似乎没看到……还好爸妈看到了没说什么 捂脸。

然后 C 面算是全尺寸键盘吧……这年头要找一个键帽上的字不是红色的怎么就这么困难呢……

过了一天收到了 SSD ，可以拆开来装上了。看一个拆机视频里说电池下面三个螺丝拧开来要用巧劲……

去路边家电维修部花了二十大洋拧了下来 手动再见 。

接下来可以拆 SSD 的包装然后安装上去啦～


关于跑分

比较偷懒就简单验了一下参数……至于质量问题现在也查不出来吧……



磁盘性能稍微有一点奇怪。内置硬盘是 5400 转的，所以速度也就一般性；但是测试下来结果非常奇怪。

SSD 性能还是可以的；希望我在软件配置上没出什么偏差（


这两天用下来的感受就是……键盘背光能不能不要默认开着……进灰能不能不要这么多……好想把屏幕上的膜撕掉（贴的时候贴坏了）……

如果对于神船有疑问的旁友欢迎私戳我。
