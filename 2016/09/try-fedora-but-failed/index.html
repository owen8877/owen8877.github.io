Fedora游记

2016-09-23
Linux

其实固定链接已经剧透了一切……

事情是这样的：因为无法忍受ubuntu的众多bug，还有上一次/home坏掉带来的影响，我决定重装系统，顺便换个发行版。

换什么好呢？好像周围的码农都在用Fedora，那我也用用好了。

于是踏上了一条不归路（微笑）。

一开始烧录安装盘的时候……居然dd和UltraIso做出来的LiveCD都没法启动……一定要用NetBSD的工具制作……奇(mo)怪(fa)。

安装的过程其实还算是满意的，就是分区挂载点配置稍微有点不直观（比起ubuntu来说）（Arch Linux在一旁偷笑）。然后安装完系统，剩下就是/home备份数据的迁移工作了。

且慢！首先要安装上一些软件试用一下吧！！！

xdroid[~]$ pip install shadowsocks
bash: pip: command not found

微笑。

xdroid[~]!$ sudo dnf install pip

We trust you have received the usual lecture from the local System
Administrator. It usually boils down to these three things:

    #1) Respect the privacy of others.
    #2) Think before you type.
    #3) With great power comes great responsibility.

[sudo] password for xdroid:

哦，还有这回事情啊，倒是第一次看见这种提示。

好，密码输入完了，接下来呢？

Fedora 24 - x86_64 - Updates    1.6 MB/s |  16 MB     00:09    
Fedora 24 - x86_64              5.1 MB/s |  47 MB     00:09    
Last metadata expiration check: 0:00:35 ago on Sun Sep 18 16:28:08 2016.
No package pip available.
Error: Unable to find a match.

心疼我自己。

可以，这很Fedora。

xdroid[~]$ sudo dnf install python-pip
xdroid[~]$ sudo pip install shadowsocks
#...
xdroid[/etc]$ sudo vim rc.d/rc.local
sudo: vim: command not found
xdroid[/etc]!$ sudo nano rc.d/rc.local
sudo: nano: command not found
xdroid[/etc]!$ sudo gedit rc.d/rc.local

行，你萌，说什么都对。

这样弄好代理总算可以安心安装chrome了。

xdroid[~]$ sudo dnf search chrome
No package chrome available.
Error: Unable to find a match.

其实应该是没有的，要像ubuntu下面那样手动添加源。

chrome终于安装完了，然而

xdroid[~]$ google-chrome --version
Google Chrome 55.0.2859.0 dev

……

还有我亲爱的vim呢？

xdroid[~]$ sudo dnf install build-essentials
Fedora 24 - x86_64 - Updates                    2.9 MB/s |  16 MB     00:05    
Last metadata expiration check: 0:00:11 ago on Mon Sep 19 22:10:35 2016.
No package build-essentials available.
Error: Unable to find a match.

正确做法是

sudo dnf install gcc-c++ make vim

然而我最终抛弃Fedora的原因是，为了能调整屏幕的背光（对，/sys/class/backlight下面什么都没有，而不是上次在ubuntu下面遇到的问题），我只好试着加一些玄学grub参数。结果重启一下，就再也无法进入图形界面了。我也无法确定是不是因为重启之前同时卸载了gnome-software的缘故。

我问我那个用Fedora的朋友。
我：Fedora的bug怎么这么多啊。
他：Fedora本来就是保持各部件最新的呀，所以bug当然很多。
我：……友尽。
